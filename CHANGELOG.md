# Changelog

Todos os registros de alterações notáveis para este projeto serão documentados neste arquivo.

O formato baseia-se em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/), e este projeto adota o [Versionamento Semântico](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2026-05-27

### Adicionado
- Inicialização do projeto base GovTech.
- Criação do arquivo `README.md` com documentação sobre o sistema.
- Configuração do sistema de padronização de commits.
  - Husky (git hooks).
  - Commitlint (validação com padrão Conventional Commits).
  - Commitizen (prompt interativo de commits).
- Histórico e tabela de versão inicial.
